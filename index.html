Habilidades

HTML

Uma linguagem de marcação de códigos que permite configurar as páginas que são veiculadas na internet. Com o HTML é possível criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

CSS

Complementando o HTML, o CSS serve para a otimização da página permitindo uma apresentação mais amigável para o usuário agregando um estilo à informação.

JavaScript

Uma linguagem de programação que vem a complementar o HTML e o CSS gerando interatividade entre cliente e servidor, ou seja, responsável pelo dinamismo da página

Git

O Git é um software de controle de versão usado para controlar o histórico de alterações de arquivos.

GitHub

Complementando o sistema Git, o GitHub é uma plataforma de hospedagem desses arquivos com controle de versão. Ele permite que desenvolvedores façam mudanças em projetos compartilhados mantendo um registro de seu progresso.
